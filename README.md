# Responsive Landing Page

A modern and responsive landing page built using HTML, CSS, and JavaScript. The project features an interactive navigation menu that changes color when scrolling and includes hover effects for better user interaction.

## Features
- Responsive Design
- Fixed Navigation Bar
- Color Change on Scroll
- Hover Effects on Menu Items
- Hero Section
- Features Section
- About Section
- Contact Form
- Smooth Scrolling

## Technologies Used
- HTML5
- CSS3
- JavaScript
