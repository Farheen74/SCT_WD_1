# Responsive Landing Page

A modern and responsive landing page built using HTML, CSS, and JavaScript. The project features an interactive navigation menu that changes color when scrolling and includes hover effects for better user interaction.

## Features
- Responsive Design
- Fixed Navigation Bar
- Color Change on Scroll
- Hover Effects on Menu Items
- Hero Section
- Features Section
- About Section
- Contact Form
- Smooth Scrolling

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run (Development)

1. (Optional) Create and activate a virtual environment:

```bash
python -m venv .venv
.venv\Scripts\activate    # Windows PowerShell
```


2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Start the Flask server:

```bash
python app.py
```

4. The site will open automatically in a new browser window at `http://127.0.0.1:5000/`.

You can still open `index.html` directly or use VS Code Live Server for static previews.
